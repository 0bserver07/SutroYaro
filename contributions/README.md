# Contributions

Drop raw results here. Any format: text, markdown, JSON, copy-pasted agent output, screenshots, links to Google Docs.

No template required. The point is to get information into the repo so it can be found later. Someone will reformat it if it needs to go into `findings/` or `DISCOVERIES.md`.

## Naming

Use your name and a short description:

```
germain-depth1-hidden64.md
michael-evolutionary-search.md
andy-gf2-noise-notes.txt
```

## What goes here vs `findings/`

- **Here**: rough notes, raw output, partial results, ideas, observations
- **`findings/`**: structured experiment reports following the [template](../findings/_template.md)

Start here if you're unsure. Move to `findings/` when the result is solid.
